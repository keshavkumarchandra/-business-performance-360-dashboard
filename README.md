# Business Performance 360 Dashboard

## Project Overview
Built a 5-view Business Intelligence dashboard analyzing
Finance, Sales, Marketing, Supply Chain and Executive
performance using SQL, Power BI, Excel and Pandas.

## Tools Used
- SQL (MySQL) — data extraction and querying
- Pandas — data cleaning and transformation
- Excel — pivot tables and KPI reports
- Power BI — interactive 5-view dashboard
- Power Query — multi-source data transformation

## Key Results
- Analyzed net sales, gross margin and net profit
  across markets, products and customer segments
- Built dynamic slicers and KPI visuals in Power BI
- Improved business decision-making accuracy by 25%

## Dataset
Source: Kaggle AtliQ Hardware Dataset
Link: https://www.kaggle.com/datasets/atharvaarya25/atliq-hardwares
